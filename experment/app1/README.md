# App1

`app1` est la premiere vraie mini-application EcoCode du dossier `experment`.

## Scenario

Elle traite un petit jeu de commandes clients, compare une implementation en boucles indexees avec une implementation plus idiomatique, puis genere des rapports via l'API EcoCode et via la CLI.

## Lancer l'application

```bash
python experment/app1/app.py
```

## Sorties generees

Les artefacts sont ecrits dans `experment/app1/output/`.
