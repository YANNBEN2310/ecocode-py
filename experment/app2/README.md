# App2

`app2` est la deuxieme mini-application EcoCode du dossier `experment`.

## Scenario

Elle analyse une liste de documents texte, compare un comptage de mots en boucles indexees avec une implementation plus idiomatique, puis produit des rapports API et CLI.

## Lancer l'application

```bash
python experment/app2/app.py
```

## Sorties generees

Les artefacts sont ecrits dans `experment/app2/output/`.
